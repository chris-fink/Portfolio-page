This was my first expreience with backend development and working in Node and ExpressJS. While the styling is minimalistic, the exercise for me was to become more comfortable operating with server-side JavaScript and using a public API to collect user inputs, and send JSON data instead of retrieving it to display.

I used MailChimp to create a database for user sign-up to collect their name and email address. Then using their API I would send the user information to update the database automatically when the user clicked the Signup button.

I also created signup success and signup failure pages that uses the https code returned by the API upon signup to determine if signup was a success or failure and redirect the user to the corresponding page. If successful, the user was taken to a page that informed them of their successful signup and prompted them to explore the London App Brewery page further. If the signup failed, the user is directed to a page that notifies them that there was an issue and invites them to try again by pressing the button on screen. This button then cycles the user back to the signup page to repeat the process.

Please note that due to the mailchimp API security the GitHub repository must be private. If you would like I would be glad to share the source code with you upon request.
